# AWS Deadline Cloud for KeyShot Development

## `hatch` commands
To develop this package, you'll need Python with `hatch` installed to start.

### Build the package

```bash
hatch build
```

### Run tests

```bash
hatch run test
```

### Run linting

```bash
hatch run lint
```

### Run formatting

```bash
hatch run fmt
```

## Run tests for all supported Python versions

```bash
hatch run all:test
```
